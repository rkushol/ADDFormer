## Folder structure
Keep your MRI data (.nii files) inside `train/test/valid` folders according to the following structure. 

data  
├── ADNI    
├  ├── AD_train   
├  ├── AD_test    
├  ├── AD_valid   
├  ├── CN_train   
├  ├── CN_test   
├  ├── CN_valid   
├  ├── AD_train_2D   
├  ├── AD_test_2D    
├  ├── AD_valid_2D   
├  ├── CN_train_2D   
├  ├── CN_test_2D   
├  ├── CN_valid_2D
